## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## Stack
- `NextJS`
- `PlanetScale` database-as-a-service
- `Prisma` ORM
- `tRPC` for type-safety between client and server
- `Radix-ui` unstyled accesible component library
- `Radix-ui/colors` color-palettes
- `Stitches` CSS in JS
